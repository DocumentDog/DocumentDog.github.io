data("ny_noaa")

ny_noaa = 
  ny_noaa |> 
  mutate(year = year(date)) |> 
  mutate(month = month(date)) |> 
  mutate(day = day(date))


ny_noaa |> 
  filter(id == "USW00014743") |> 
  filter(year > 2009) |> 
  plot_ly(
    x = ~snwd, y = ~tmin, type = "scatter", mode = "markers"
  )
## Warning: Ignoring 34 observations

From this scatterplot, we can see that the snow depth does not have relationship with minimum temperature. Also, scatterplot might not be a good plot to show the relationship between these two variables.

ny_noaa |> 
  filter(id == "USW00014743") |> 
  plot_ly(
    x = ~month, y = ~snwd, type = "box"
  )
## Warning: Ignoring 151 observations

From this boxplot, it is clear that in the area for this station, snow usually falls between December to March, and the deepest snow depth is in February.

ny_noaa |> 
  filter(id == "USW00014743") |> 
  group_by(year) |> 
  summarize(avg_prcp = mean(prcp, na.rm = TRUE)) |> 
  plot_ly(
    x = ~year, y = ~avg_prcp, type = "bar"
  )

By this barplot, we can see the average precipitation in for each year in this specific area. Precipitation in this region has a three-to-four year cycle: the precipitation is normal at first but gradually increase at last.
